====== Download drivers for Frankfurt RS-232 ======

You can use the Frankfurt RS-232 with your own code either by writing code for the [[http://www.vscp.org/docs/vscpd/doku.php?id=level1_driver_can4vscp|CAN4VSCP serial driver]] which expose a [[http://www.vscp.org/docs/vscpd/doku.php?id=canal_interface_specification|CANAL interface]] (AKA Level I interface) or written directly to interface one of the modes of the Frankfurt RS-232 module ([[the_verbose_mode|verbose]], [[the_can4vscp_mode|CAN4VSCP]] or [[the_slcan_mode|SLCAN]]).

The CAN4VSCP is included in the [[https://sourceforge.net/projects/m2m/files/VSCP%20Software/|VSCP & Friends package]].
